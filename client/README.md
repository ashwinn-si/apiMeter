# ApiMeter - Client

This repository contains the frontend client application for the ApiMeter project.

## Features
- User dashboard for managing API keys
- Viewing usage statistics and invoices
- Selecting and upgrading subscription tiers

## Configuration
> [!IMPORTANT]
> The `.env` file is excluded from version control for security. To run the client locally, you must create it manually.

1. Copy the provided `.env.example` file to create your own `.env` file:
   ```bash
   cp .env.example .env
   ```
2. Update the variables inside `.env` if your backend is running on a different URL.
