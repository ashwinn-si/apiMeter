# ApiMeter - Eureka Server

This repository contains the Service Discovery registry for the ApiMeter project, powered by Netflix Eureka.

## Features
- Service Registration: Microservices register themselves upon startup.
- Service Discovery: Allows services to find and communicate with each other dynamically.

## Configuration
> [!IMPORTANT]
> The `application.yml` / `application.yaml` files have been intentionally excluded from version control for security. To run this service, create a new `application.yaml` inside `src/main/resources/` and add the necessary configurations for your environment.
